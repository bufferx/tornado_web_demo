twork
==========

twork is a web server framework based on tornado.

Usage
------------
* cd ./src/www/
* python service.py -env=debug

Test
------------
* http://localhost:8000/sayhi?name=bufferx

Requirements
------------
The following libraries are required

* [tornado](http://github.com/facebook/tornado)

Issues
------
